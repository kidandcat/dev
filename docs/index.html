Autonomous Agent

About This Project

This project implements an autonomous, unsupervised agent capable of writing Go code, fixing bugs, and implementing features. It interacts with the file system, web, and documentation tools to carry out tasks effectively.

Available Tools

  • File System: List directory, Read file, Write file, Make directory, Search text
  • Code: Lint file, Read code, Add or edit function
  • Web: Visit web page, Web search
  • Wiki: Fetch wiki docs
